Overview:-<br>
This project is a Python-based Command Line Interface (CLI) application designed to automate the calculation of semester CGPA following VIT Bhopal University guidelines. It supports multiple course types with distinct grading schemes, credits, and attendance considerations, providing students with instant, accurate CGPA results.

Features:- <br>
•	Supports various course types: LT, LTP, LP, P, PJ<br>
•	Calculates grades based on university-specific schemes<br>
•	Input validation for course details, marks, attendance<br>
•	Handles multiple courses in one semester<br>
•	Calculates weighted CGPA based on course credits and grade points<br>
•	Modular code structure for easy maintenance and extension<br>


Technologies / Tools Used:-<br>
•	Python 3<br>
•	Jupyter  Notebook (for interactive use and demonstration)<br>
•	Git and GitHub for version control and submission<br>


Steps to Install & Run:-<br>
1.	Ensure Python 3 is installed on your machine.<br>
2.	Clone or download this repository from GitHub:<br>
bash<br>
git clone https://github.com/coderfrommau/vit-cgpa-calculator.git<br>
3.	Navigate into the folder:<br>
bash<br>
cd vit-cgpa-calculator<br>
4.	Launch the Jupyter Notebook:<br>
bash<br>
jupyter notebook main_notebook.ipynb<br>
or run the standalone script if you have main.py.
How to Test<br>
•	Open  main_notebook.ipynb in Jupyter.<br>
•	Run all cells step-by-step.<br>
•	Enter course details, marks, attendance as prompted.<br>
•	View the CGPA output at the end.<br>
